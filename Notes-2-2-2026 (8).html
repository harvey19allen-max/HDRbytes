Introduction to Coding & Programming

    
Coding is the structured process of writing instructions that computers can interpret and execute.

    
These instructions allow machines to solve problems, automate tasks, and control physical systems such as robots.

    
Programming applies logic, mathematics, and problem-solving to create efficient digital solutions.

    
In education and research, coding supports innovation, critical thinking, and technological advancement.

Basic Coding Concepts (Robotics-Oriented)

    

      
      

        
Numbers

        
Represent numerical values.

        

          
Numbers store quantities such as distance, speed, or sensor values.

          
int distance = 25;
float temperature = 36.5;
          
        

      


      
      

        
Strings

        
Store text data.

        

          
Strings hold words, labels, or messages.

          
String message = "Hello, Robot!";
          
        

      


      
      

        
Booleans

        
Represent true or false values.

        

          
Booleans control logic decisions.

          
boolean isActive = true;
          
        

      


      
      

        
AND Operator

        
Combines conditions.

        

          
The AND operator ensures multiple conditions are true.

          
if(sensor1 && sensor2) {
  moveForward();
}
          
        

      


      
      

        
Variable Declaration

        
Creates a variable.

        

          
Declaring a variable reserves memory.

          
int speed;
          
        

      


      
      

        
Variable Definition

        
Assigns a value.

        

          
Defines how the variable is used.

          
speed = 10;
